### How to Run

#### Prerequisites

You need to have docker with compose plugin or old docker-compose and npm installed.

##### Run Backend

First spin up the backend service:
```bash
$> cd backend
$> docker compose up
```

##### Run UI
```bash
$> cd frontend
$> npm run start
```

##### Run Test (currently only for UI)
```bash
$> cd frontend
$> npm run test
```
